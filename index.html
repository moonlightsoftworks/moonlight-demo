Moonlight

A sleek GUI tool to check if a Roblox username is online, in a game, or using Studio.

Features

  • ✔ Real-time Roblox username status detection
  • ✔ Detects if user is in a game or using Studio
  • ✔ Sleek and draggable UI
  • ✔ Toggle visibility with the "V" key
